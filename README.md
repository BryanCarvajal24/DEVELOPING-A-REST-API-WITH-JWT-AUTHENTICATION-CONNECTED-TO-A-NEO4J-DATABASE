# DEVELOPING-A-REST-API-WITH-JWT-AUTHENTICATION-CONNECTED-TO-A-NEO4J-DATABASE

## Descripción
The development of this project involves creating a REST API in Visual Studio Code with JWT (JSON WEB TOKEN) authentication, managing data related to players and their teams, connected to a Neo4j (Graph Database) using Neo4j Desktop for database management.

El desarrollo de este proyecto implica la creación de una API REST en Visual Studio Code con autenticación JWT (JSON WEB TOKEN), gestionando datos relacionados con jugadores y sus equipos, conectada a una base de datos Neo4j (base de datos orientada a grafos) utilizando Neo4j Desktop para la gestión de la base de datos.

## Instalación
Este proyecto requiere la instalación de Node.js, un editor de código como Visual Studio Code y una base de datos Neo4j Desktop. Posteriormente, configura y ejecuta la API REST conectándola a la base de datos Neo4j para su correcto funcionamiento.
